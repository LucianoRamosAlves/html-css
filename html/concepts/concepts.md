
## Estrutura Básica

```html
<!DOCTYPE html>
<html lang="pt-BR">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Página</title>
    </head>
    <body>
        <header></header>
        <main></main>
        <footer></footer>
    </body>
</html>
```

## Elementos Semânticos

- `<header>` - Cabeçalho da página
- `<nav>` - Navegação
- `<main>` - Conteúdo principal
- `<article>` - Artigo independente
- `<section>` - Seção temática
- `<aside>` - Conteúdo complementar
- `<footer>` - Rodapé

## Elementos de Texto

- `<h1>` a `<h6>` - Títulos
- `<p>` - Parágrafo
- `<strong>` - Texto importante
- `<em>` - Ênfase
- `<a>` - Link

## Listas

- `<ul>` - Lista desordenada
- `<ol>` - Lista ordenada
- `<li>` - Item da lista
