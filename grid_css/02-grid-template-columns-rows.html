📐 grid-template-columns e grid-template-rows

grid-template-columns e grid-template-rows são as propriedades que definem a estrutura do grid.


📌 grid-template-columns → Define quantas colunas e seus tamanhos
📌 grid-template-rows → Define quantas linhas e seus tamanhos


Cada valor na lista representa uma coluna/linha. Exemplo:
grid-template-columns: 100px 200px 100px = 3 colunas (100px, 200px, 100px)

1️⃣ repeat(4, 1fr) - 4 colunas iguais

grid-template-columns: repeat(4, 1fr);
O repeat() repete um valor N vezes. Aqui: 4 colunas de 1fr cada.
✨ Dica: Use repeat() sempre que precisar de colunas/linhas repetidas!

2️⃣ auto-fill - GRID RESPONSIVO MÁGICO

grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));
🔥 Esse é um dos padrões mais úteis do Grid!
auto-fill cria automaticamente quantas colunas couberem no container.
minmax(150px, 1fr) = cada coluna tem no mínimo 150px, no máximo 1fr.

Redimensione a tela para ver a mágica acontecer! ✨

3️⃣ minmax() - Flexibilidade controlada

grid-template-columns: minmax(100px, 300px) minmax(200px, 500px);
2 colunas onde:
- Coluna 1: mínimo 100px, máximo 300px
- Coluna 2: mínimo 200px, máximo 500px
O navegador ajusta dentro desses limites conforme o espaço disponível

100px ~ 300px
200px ~ 500px

4️⃣ auto - Tamanho pelo conteúdo

grid-template-columns: auto 2fr auto;
auto faz a coluna ter o tamanho exato do conteúdo.
Útil para colunas que devem se adaptar ao conteúdo (ex: imagens, texto)

5️⃣ Misturando unidades

grid-template-columns: 100px 30% 1fr;
Podemos misturar px, %, fr, etc.
1ª coluna: 100px fixos | 2ª coluna: 30% do container | 3ª coluna: resto (1fr)

6️⃣ grid-template-rows - Alturas definidas

grid-template-rows: 80px 150px 100px;
Define a altura de CADA LINHA individualmente.
Linha 1: 80px | Linha 2: 150px | Linha 3: 100px

🔹 Se houver mais linhas que valores definidos, elas se ajustam ao conteúdo
