📐 gap, row-gap, column-gap

As propriedades de gap controlam o espaçamento interno entre as células do grid. Elas NÃO adicionam espaço nas bordas externas do container!


📌 row-gap → Espaço entre as linhas (vertical)
📌 column-gap → Espaço entre as colunas (horizontal)
📌 gap → Shorthand: gap: row-gap column-gap


⚠️ Importante: Antes se usava grid-gap, grid-row-gap, etc.
Hoje as versões sem grid- são padrão e funcionam também no Flexbox!

1️⃣ SEM gap - Células grudadas

gap: 0; — Sem espaçamento, as células ficam coladas umas nas outras.

📝 Resumo

gap: 10px; → 10px entre linhas E colunas
gap: 10px 20px; → 10px entre linhas, 20px entre colunas
row-gap: 10px; → Só espaço entre linhas
column-gap: 20px; → Só espaço entre colunas

🔹 Use gap (shorthand) na maioria dos casos para código mais limpo!
