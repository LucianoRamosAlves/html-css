📐 CSS Grid Layout

O que é CSS Grid?

CSS Grid Layout é um sistema de layout bidimensional criado especificamente para organizar elementos em linhas e colunas simultaneamente.


🔹 Flexbox = 1 dimensão (ou linha OU coluna)
🔹 Grid = 2 dimensões (linhas E colunas ao mesmo tempo)

Exemplo 1: Grid com tamanhos fixos

Usamos grid-template-columns: 100px 100px 100px para criar 3 colunas de 100px cada.
Usamos grid-template-rows: 100px 100px para criar 2 linhas de 100px cada.
Total: 6 células (3 colunas × 2 linhas)

Exemplo 2: Grid com frações (fr)

A unidade fr (fraction) divide o espaço disponível proporcionalmente.
grid-template-columns: 1fr 2fr 1fr = 4 partes no total.
Coluna 1 = 1/4, Coluna 2 = 2/4 (metade), Coluna 3 = 1/4 do espaço.

📝 Dica: Use fr para layouts responsivos sem media queries!
