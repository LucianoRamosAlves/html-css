📐 grid-area e grid-template-areas

grid-template-areas permite criar layouts usando nomes visuais em formato de "mapa".
É a forma MAIS INTUITIVA de criar layouts com Grid!


⚡ grid-template-areas → Define o mapa no CONTAINER
⚡ grid-area → Associa o item à área no ITEM FILHO

1️⃣ Layout clássico com grid-template-areas

Mapa do layout:
"header header header"
"nav main aside"
"footer footer footer"

🔹 Cada string = uma linha do grid
🔹 Cada nome dentro da string = uma coluna
🔹 Nomes iguais em células adjacentes = uma área só
🔹 O formato da área deve ser retangular

HEADER
NAV
MAIN (conteúdo principal)
ASIDE
FOOTER

2️⃣ Células vazias com "."

Mapa do layout com células vazias:
"header header ."
"nav main aside"
". footer footer"

O ponto (.) representa uma célula que existe mas está VAZIA.
Útil para criar layouts assimétricos ou pular posições.

3️⃣ Layout RESPONSIVO com grid-template-areas

O grande poder do grid-template-areas: Mudar o layout completamente em diferentes tamanhos de tela, apenas alterando o mapa!

Redimensione a tela para ver o layout mudar:

🖥️ Desktop: 4 colunas (nav | main | main | aside)
📱 Tablet: 2 colunas (main + aside)
📱 Celular: 1 coluna (tudo empilhado)

✨ SEM PRECISAR ALTERAR O HTML!

NAV
MAIN - Conteúdo Principal
ASIDE - Sidebar
FOOTER

4️⃣ grid-area como shorthand

grid-area também pode ser usado como shorthand para:
grid-row-start / grid-column-start / grid-row-end / grid-column-end

📌 grid-area: 1 / 1 / 3 / 2;
🔹 row-start: 1
🔹 col-start: 1
🔹 row-end: 3
🔹 col-end: 2

⚠️ Memorize: A ordem é ROW depois COLUMN!
(linha/coluna/linha/coluna) - difícil mas útil!

grid-area:
1 / 1 / 3 / 2
2
3
4
grid-area:
1 / 3 / 2 / 4
6

📝 Regras do grid-template-areas

✅ Nomes iguais em células adjacentes = formam UMA área
✅ Células vazias usam . (ponto)
✅ A área deve ser retangular (não pode ser "L" ou "T")
✅ Todas as linhas devem ter o mesmo número de colunas
✅ Use media queries para redefinir o layout em diferentes telas

🔥 Dica de ouro: grid-template-areas + media queries = layout responsivo sem tocar no HTML!
